Need These Slide 2 of 6

Before You Start

Have four things ready.

1. Python environment

Create the repo environment first so the openai package and scripts are available.

2. TAMU API key

Your key goes into TAMU_API_KEY. Keep it private.

3. TAMU Azure endpoint

Put the endpoint in TAMU_AZURE_ENDPOINT. This repo uses that path when it is present.

4. Right API mode

Use API_MODE=chat_completions. That is the path verified in this repo.

If your instructor gave you a deployment name, you can also set TAMU_DEPLOYMENT.
Setup Slide 3 of 6

Environment

Open the project and install dependencies.

cd LLM_PV
conda env create -f environment.yaml
conda activate llm_pv

If you already made the environment, just conda activate llm_pv.

The commands in these slides assume your terminal is inside LLM_PV.
.env Slide 4 of 6

Configuration

Create .env in LLM_PV.

TAMU_API_KEY=your_key_here
TAMU_AZURE_ENDPOINT=https://your-resource.openai.azure.com/
TAMU_API_VERSION=2024-12-01-preview
OPENAI_MODEL=protected.gpt-5.2
API_MODE=chat_completions

The repo already maps protected.gpt-5.2 to the tested deployment alias gpt-5.2-deep-learning-fundamentals.

Do not commit this file.
Test Slide 5 of 6

Smoke Check

Run the smallest possible test first.

python program_synthesis/boosted/tamu_api_smoke.py --timeout 90

You want output that includes "ok": true and "client_type": "azure_openai".

Tested result in this repo: the smoke test returned "ok": true.
If you try responses mode here, the current SDK/client combo is not the safe default. Stay on chat_completions.
Next Step Slide 6 of 6

Real Run

Then launch one tiny verified job.

python program_synthesis/runner.py --functions fn_a --lengths 20 --attempts 1 --num-trials 1 --train-size 8 --val-size 8 --test-size 32 --timeout 120
  1. If this works, your TAMU setup is good.
  2. If you get a key error, fix TAMU_API_KEY.
  3. If you get a 401 or 403, your TAMU account probably does not have access yet.
  4. If you get a 404, check the endpoint or deployment name.
This exact command completed successfully in this repo during verification.
Fast rule: smoke test first, tiny runner second, big experiments last.
